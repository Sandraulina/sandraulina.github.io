Sistem Informasi Laboratorium

Sistem Informasi Laboratorium merupakan website berbasis html, css, bootstrap, javascript, dan php dengan framework laravel. Proyek ini dikerjakan secara berkelompok yang beranggotakan 5 orang, dimana saya berperan sebagai full stack developer. Berikut merupakan tampilan login dari sistem informasi laboratorium:


Setelah berhasil login, maka akan menampilkan halaman berikut:


Menu transaksi berfungsi untuk menampilkan daftar transaksi


Menu peralatan berfungsi untuk menampilkan daftar alat yang tersedia pada laboratorium


Menu riwayat aktivitas berfungsi untuk menampilkan daftar riwayat aktivitas pada sistem informasi laboratorium


Menu request berfungsi untuk menampilkan daftar alat yang direquest oleh mahasiswa. Laboran juga dapat mengirim pesan kepada mahasiswa tersebut


Menu viev alat terdapat pada halaman mahasiswa dan berfungsi untuk menampilkan daftar alat yang dapat direquest oleh mahasiswa


Menu daftar request terdapat pada halaman mahasiswa dan berfungsi untuk menampilkan daftar request mahasiswa serta dapat berkomunikasi dengan laboran melalui chat
